# 🚆 Train Ticket & Revenue System

## 📌 Project Overview
The Train Ticket & Revenue System is a Python-based railway management application designed to handle passenger ticket booking, fare calculation, seat allocation, cancellation, and revenue analytics. The system integrates MySQL for database management and Matplotlib for graphical data visualization.

---

## 🛠️ Tech Stack
- **Programming Language:** Python
- **Database:** MySQL
- **Libraries Used:**
  - Pandas
  - Matplotlib
  - mysql-connector-python

---

## ✨ Features
- Passenger Ticket Booking
- Dynamic Fare Calculation
- Live Seat Availability
- Automatic Seat Allocation
- Ticket Cancellation & Refund Logic
- Revenue Management System
- Database Storage using MySQL
- Revenue Analytics using Charts
- Monthly Revenue Analysis

---

## 📊 Visualizations
The project generates:
- Bar Charts for class-wise revenue
- Pie Charts for revenue distribution
- Revenue trend analysis graphs

---

## 🗄️ Database Integration
The system stores:
- Passenger Details
- Route Information
- Ticket Fare
- Booking Date
- Seat Numbers

using MySQL database integration.

---

## 🚉 Available Routes
- SEC to Rayalaseema

## ▶️ How to Run the Project

### Step 1: Install Required Libraries

```bash
pip install pandas matplotlib mysql-connector-python
```

### Step 2: Create MySQL Database

```sql
CREATE DATABASE railwaydb;
```

### Step 3: Update Database Credentials
Edit the MySQL username and password in the Python file.

### Step 4: Run the Project

```bash
python PRO3.py
```
